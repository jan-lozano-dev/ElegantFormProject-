# Elegant Form Builder

A modern, responsive web form built with AI assistance to explore the potential of AI-driven development workflows.

## 🎯 Project Overview

This project showcases a clean, elegant form interface developed as an exploration into AI-assisted web development. Built using Claude Code, it demonstrates how AI can accelerate the development process while maintaining code quality and design standards.

## ✨ Features

- **Responsive Design** - Optimized for all device sizes
- **Modern UI/UX** - Clean, intuitive interface
- **Form Validation** - Client-side validation for better UX
- **Accessible** - Built with accessibility best practices
- **Fast Loading** - Optimized performance

## 🚀 Technologies Used

- HTML5
- CSS3 (with modern features)
- JavaScript (ES6+)
- Responsive design principles

## 🛠️ Installation & Setup

1. Clone the repository:
   ```bash
   git clone https://github.com/jan-lozano-dev/simple_event.git
   cd simple_event
   ```

2. Open in your preferred development environment or simply open `index.html` in a web browser.

3. For local development with live reload, you can use:
   ```bash
   # Using Python's built-in server
   python -m http.server 8000
   
   # Or using Node.js
   npx live-server
   ```
